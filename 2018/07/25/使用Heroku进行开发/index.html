  • Heroku 是 Salesforce 旗下云服务商，提供方便便捷的各种云服务，如服务器，数据库，监控，计算等等,Heroku 平台的灵活性极高且支持多种编程语言。若想把程序部署到 Heroku 上,开发者要使用 Git 把程序推送到 Heroku 的 Git 服务器上。并且他提供了免费版本.对于个使用,搞一些小程序还是可以满足使用的.
